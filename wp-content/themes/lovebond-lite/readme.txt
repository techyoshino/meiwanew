== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Lovebond Lite in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
		Or
1. Download the latest version from https://wordpress.org/themes/lovebond-lite/
2. Go to Appearance >> Add New >> Upload Theme.
3. Select the downloaded lovebond-lite.zip file and click Install Now.


== Customization ==
1. Navigate to Appearance > Customize in your admin panel and customize.


== Frequently Asked Questions ==

= How to set Theme Front Page? =
1. Select "A static page" to show on from "Settings -> Reading -> Front page displays"

2. To Write Page Content
	Go to "Appearance -> Cusomize -> Home Landing Page Section"
	Add landing page section content here and follow below instructions for setting menu.

3. To set the Navigation
	Go to "Appearance -> Menus"
	>>For First Section :- Create a custom link in Menus and put #section1 in URL and Navigation Label according to you for landing page.
	>>For Second Section :- Create a custom link in Menus and put #section2 in URL and Navigation Label according to you for landing page.
	>>For Third Section :- Create a custom link in Menus and put #section3 in URL and Navigation Label according to you for landing page.
	>>For Home Blog Section :- Create a custom link in Menus and put #front-posts-box in URL and Navigation Label according to you for landing page.

4. To change second section background image.
	Use child theme or any custom css plugin and write below css code replace the image url parameter with new image

	#section2:before{background-image: url('images/section-bg.jpg');}

= How to use theme in different languages? =
The Lovebond Lite is ready for translation. Translation files are in /languages.
Read the following blog for more https://sketchthemes.com/blog/effective-steps-to-translate-your-wordpress-theme-using-qtranslate/
